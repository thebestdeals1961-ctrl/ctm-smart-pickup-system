# 🚕 CTM Smart Pickup System

A web-based taxi dispatch SaaS platform built with Node.js, React, and PostgreSQL (Supabase).

## Features

- ✅ **User Authentication** - Secure JWT-based login system
- ✅ **Role-Based Access** - Admin, Dispatcher, and Driver roles
- ✅ **Ride Management** - Create, assign, track, and complete rides
- ✅ **Driver Management** - Online/offline status, vehicle tracking
- ✅ **Passenger Management** - Passenger profiles and ride history
- ✅ **Dashboard** - Real-time statistics and analytics
- ✅ **Responsive Design** - Works on desktop and mobile

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Node.js + Express |
| Frontend | React + React Router |
| Database | PostgreSQL (Supabase) |
| Auth | JWT + bcryptjs |
| Deployment | GitHub + Vercel compatible |

## Quick Start

1. **Set up database** → See `docs/DATABASE_SCHEMA.md`
2. **Run backend** → `cd backend && npm install && npm run dev`
3. **Run frontend** → `cd frontend && npm install && npm start`

Full instructions in `docs/SETUP_GUIDE.md`.
